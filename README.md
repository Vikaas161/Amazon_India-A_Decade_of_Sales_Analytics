# amazonEDA
Amazon India: A Decade of Sales Analytics-
I built an end-to-end analytics platform using ten years of Amazon India sales data to show how a major e-commerce business actually runs. I started with a massive, 'messy' dataset where one out of every four entries had errorsâ€”things like broken dates, weird currency symbols, duplicate orders, handling outliers.
